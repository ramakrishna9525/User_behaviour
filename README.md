# Customer Shopping Behavior Analysis

Customer Shopping Behavior Analysis is a data analytics project designed to explore and understand customer purchasing patterns, preferences, and business trends using real-world retail transaction data. The project leverages Python for data cleaning, preprocessing, exploratory data analysis (EDA), and feature engineering, PostgreSQL for structured data storage and advanced SQL querying, and Power BI for interactive dashboard creation and business intelligence reporting.

The dataset contains customer demographic information, product details, purchase history, review ratings, subscription status, shipping preferences, discount usage, and transaction amounts. Through comprehensive analysis, the project identifies key insights such as revenue contribution by customer segments, top-performing products, customer loyalty patterns, subscription behavior, discount effectiveness, and age-group purchasing trends.

Advanced SQL queries were developed to perform customer segmentation, revenue analysis, product ranking, behavioral analysis, and performance evaluation across multiple business dimensions. The results were visualized through interactive dashboards to support data-driven decision-making and strategic business planning.

This project demonstrates practical skills in data analytics, database management, business intelligence, data visualization, and SQL-based reporting. It showcases the complete analytics workflow, from raw data processing to actionable business insights, making it a valuable portfolio project for data analyst, business analyst, and data science roles.
